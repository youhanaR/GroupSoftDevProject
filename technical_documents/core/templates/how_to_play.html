How to Play:

  • Explore the Map: Click on any building on the map to interact with it.
  • Get the Access Code: Scan the QR code at the real-world location of the building to get your unique access code.
  • Enter the Code: Type the access code in the pop-up that appears when you click on the building.
  • Start the Game: If the code is correct, you'll be taken to the mini-game at that location.
  • Try Again: If the code is incorrect, an error message will appear. Double-check your QR scan and try again!
