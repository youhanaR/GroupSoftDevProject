Recycle Rush

This game is part of the Sustainability Project for the University of Exeter.

In Recycle Rush, you control a recycling bin to catch falling waste items. Your goal is to sort the waste correctly by switching between the different bins using the on-screen buttons.

How to Play

  • Use the left (⬅) and right (➡) buttons to move the bin.
  • Switch between bins using the on-screen selection buttons (Plastic, Battery, Can, Paper).
  • Catch the correct waste in the matching bin to earn points.
  • If you miss or catch the wrong item, you lose a life.
  • The game speeds up as you score more points!

Why This Matters

Recycling helps reduce waste, conserve natural resources, and minimize pollution. By playing this game, you learn the importance of sorting waste properly and how small actions can contribute to a sustainable future.
